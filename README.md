## Info Hacker API

## Technology Used:

- Nest.JS
- TypeScript
- MongoDB
- Mongoose
- Mongoose Aggregation
- JWT - Json Web Token
- Passport
- Web Socket

## Features

- Magic link authentication [link expiration time: 15 mins]
- User crud API

## Instructions to run the app

- pnpm/npm/yarn install
- pnpm/yarn dev || npm run dev
