## Info Hacker API

## Technology Used:

- Nest.JS
- TypeScript
- MongoDB
- Mongoose
- JWT - Json Web Token
- Passport

## Features

- JWT Authentication
- User crud API
- Chat crud API

## Instructions to run the app

- pnpm/npm/yarn install
- pnpm/yarn dev || npm run dev
